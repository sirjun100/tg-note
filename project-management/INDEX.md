| `docs/` | Templates and reference |

## Workflow

1. **View backlog** → [product-backlog.md](backlog/product-backlog.md)
2. **Create user story** → Use `create_user_story` tool or copy template to `backlog/user-stories/US-XXX-name.md`
3. **Create defect** → Use `create_defect` tool or copy template to `backlog/defects/DEF-XXX-name.md`
4. **Plan sprint** → Create `sprints/sprint-XX-name.md`
